# n8n-as-code Agent Skills

Portable n8n-as-code skills for agents that can read `SKILL.md` files.

## Skill

- `n8n-architect`: workspace readiness, migration, environments, managed local runtime guidance, tunnels, workflow authoring, sync, node schemas, validation, push/pull, and context-root workflow operations.

Install the skills in your agent runtime from this repository directory:

```
https://github.com/EtienneLescot/n8n-as-code/tree/main/skills
```

This directory contains the required skill:

```
skills/n8n-architect/SKILL.md
```

Generic skill installers can discover the `skills/*/SKILL.md` directories. Installers that require an explicit subpath can use:

```
skills/n8n-architect
```

Workspace-local copies generated by `n8nac update-ai` live at:

```
.agents/skills/n8n-architect/SKILL.md
```
